# Sachsen Wahl 2019 :ballot_box:

Shows word clouds of the manifestos

## How it works

* Converting pdfs to MD via [pdf2md](http://pdf2md.morethan.io/)
* Tokenizing
* Removing stopwords
* Stemming
* Creating word cloud via [wordcloud2.js](https://github.com/timdream/wordcloud2.js/)
